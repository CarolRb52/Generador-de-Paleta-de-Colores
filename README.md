                                          ## 💡 Cómo Usar

                                          1. **Selecciona un color base:**
                                             - Usa el color picker visual O
                                                - Ingresa un código hexadecimal (#RRGGBB)

                                                2. **Elige una armonía:**
                                                   - Selecciona el tipo de armonía que deseas
                                                      - La paleta se genera automáticamente

                                                      3. **Interactúa con la paleta:**
                                                         - Haz clic en cualquier color para copiarlo
                                                            - Edita colores haciendo clic en "✎ Editar"
                                                               - Visualiza nombres de colores automáticos

                                                               4. **Exporta tu paleta:**
                                                                  - CSS Variables: Para usar en proyectos web
                                                                     - Tailwind Config: Para proyectos Tailwind
                                                                        - JSON: Para cualquier aplicación

                                                                        5. **Visualiza previsualizaciones:**
                                                                           - Cambia entre vistas (Card, UI, Marketing)
                                                                              - Ve cómo se vería tu paleta en diferentes contextos

                                                                              ## 🎨 Tipos de Armonías

                                                                              ### Complementaria
                                                                              Combina colores opuestos en la rueda de colores para máximo contraste.

                                                                              ### Triádica
                                                                              Usa tres colores equidistantes en la rueda de colores, creando un esquema vibrante.

                                                                              ### Tetradica
                                                                              Cuatro colores en forma de rectángulo en la rueda de colores, equilibrado y versátil.

                                                                              ### Análoga
                                                                              Colores adyacentes en la rueda de colores, crea una armonía natural y relajante.

                                                                              ### Complementaria Dividida
                                                                              Similar a complementaria pero con dos colores adyacentes al opuesto, más flexible.

                                                                              ### Monocromática
                                                                              Variaciones del mismo color con diferentes saturación y luminosidad.
